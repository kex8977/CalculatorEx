오늘은 iOS에서 동작할 간단한 계산기 앱을 만들었다.

이를 위해 평소에 컴퓨터나 휴대폰에서 사용하던 계산기는 어떻게 생겼는지 찾아봤다.

생각보다 많은 기능을 제공하고 있었다는 걸 세삼 놀랐다.
사칙연산부터 날짜계산, 길이, 거리, 무게 등 다양한 기능을 제공했고, 메모리 기능도 있었다.

그래도 오늘 목표는 사칙연산이 가능한 계산기를 만들어 보는 것
